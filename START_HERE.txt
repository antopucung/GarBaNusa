Fix: Try: npm install --legacy-peer-deps

Problem: Port 3000 in use
Fix: Use different port: PORT=3001 npm run dev

Problem: TypeScript errors
Fix: These will disappear after npm install

Problem: Vercel deploy fails
Fix: Run npm run build locally first to test
